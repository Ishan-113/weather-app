# Weather App

A beginner-friendly weather app built with HTML, CSS, and JavaScript.

The app uses the OpenWeatherMap API to show current weather details for a city.

## Live Demo

[weather-app-chi-two-25.vercel.app](https://weather-app-chi-two-25.vercel.app/)

## Features

- Search current weather by city name.
- Display temperature in Celsius.
- Show feels-like temperature.
- Show humidity and wind speed.
- Render weather icons based on current conditions.
- Search with the Enter key or the search button.
- Use a responsive interface for desktop and mobile screens.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Fetch API
- async/await
- OpenWeatherMap API
- Vercel

## Run Locally

```bash
git clone https://github.com/Ishan-113/weather-app.git
cd weather-app
```

Open `index.html` in your browser.

If you replace the API key, update the `API_KEY` value in `script.js`.
