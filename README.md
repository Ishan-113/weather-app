# Weather App

A beginner-friendly weather app built with HTML, CSS, and JavaScript using the OpenWeatherMap API.

## Live Demo

[weather-app-chi-two-25.vercel.app](https://weather-app-chi-two-25.vercel.app/)

## Features

- Search current weather by city name.
- Display temperature in Celsius.
- Show feels-like temperature, humidity, and wind speed.
- Render weather icons based on current conditions.
- Support searching with the Enter key or search button.
- Responsive interface for desktop and mobile screens.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Fetch API and async/await
- OpenWeatherMap API
- Vercel deployment

## Run Locally

```bash
git clone https://github.com/Ishan-113/weather-app.git
cd weather-app
```

Open `index.html` in your browser. If you replace the API key, update the `API_KEY` value in `script.js`.
