# 🌤️ Weather App

A weather app built with HTML, CSS and JavaScript using the OpenWeatherMap API.

## 🚀 Live Demo
https://weather-app-chi-two-25.vercel.app/

## ✨ Features
- Search weather for any city in the world
- Shows real-time temperature in °C
- Feels like temperature
- Humidity and wind speed
- Weather icons based on conditions (☀️ ☁️ 🌧️ ⛈️)
- Press Enter or click Search to get results

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox, Glassmorphism)
- Vanilla JavaScript (fetch, async/await)
- OpenWeatherMap API
- Deployed on Vercel
